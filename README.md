
# 📦 SmartMarket

SmartMarket is a full-stack multi-vendor e-commerce web application that allows customers to buy products, sellers to manage their stores, and administrators to control the platform.

The live website can be found at:
👉 [https://smartmarket-end0.onrender.com/](https://smartmarket-end0.onrender.com/)

---

## 🚀 Overview

SmartMarket is designed to simulate a real-world online marketplace where multiple sellers can register and list their products, while customers can browse, purchase, and manage orders — all under a centralized admin system.

The system focuses on:

* Clean user experience
* Role-based access control
* Secure authentication
* Real-world e-commerce structure

---

## 👥 User Roles

SmartMarket supports three types of users:

### 🛍️ Customer

* Register and log in
* Browse available products
* Add products to cart
* Place orders
* View order history
* Manage profile information

### 🏪 Seller

* Register as a seller
* Add new products
* Edit or delete their products
* View customer orders for their products
* Manage inventory

### 👑 Admin

* Manage all users
* Manage all products
* Oversee platform activity
* Maintain system control and moderation

---

## ⚙️ Key Features

* Multi-vendor marketplace system
* Role-based authentication (Customer, Seller, Admin)
* Secure password hashing (bcrypt)
* MongoDB Atlas cloud database
* RESTful backend architecture
* Server-side rendering using EJS
* Clean and responsive UI
* Hosted and deployed online

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* EJS (Embedded JavaScript Templates)

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB Atlas

**Deployment:**

* GitHub (Version Control)
* Render (Hosting Platform)

---

## 🔐 Security

* Passwords are securely hashed using bcrypt
* Environment variables are used for sensitive data
* Role-based route protection for admin and seller access
* Database hosted securely on MongoDB Atlas
